# 雅思成绩报告字段清单

## 1. 文档说明

本附件基于当前原型页面 [雅思成绩分析.html](file:///d:/AI学习/test_center/雅思成绩分析.html) 的实际展示内容整理，用于给产品、研发和前端对齐成绩报告的数据结构与展示结构。

当前整理原则如下：

- 先按现有页面已展示内容收敛，不额外扩展未展示字段。
- 以四科 `Tab` 结构为主，梳理摘要区和明细区。
- 字段命名以语义化为主，作为后续接口与前端映射的参考命名。
- 本次仅补充 `是否必填`、`枚举值`、`示例值`。
- 涉及雅思最终换算规则、总分计算、正式报告版式的部分，暂标记为 `待确认`。

```json
{
  "examId": "string",
  "candidateId": "string",
  "reportId": "string",
  "reportType": "IELTS",
  "subjectTabs": [
    {
      "subjectCode": "listening",
      "subjectName": "听力",
      "summary": {},
      "overview": {},
      "details": []
    }
  ]
}
```

### 8.2 对齐建议

| 维度 | 建议 |
| --- | --- |
| 产品 | 先确认当前页面是否作为一期正式展示基线 |
| 前端 | 尽量按科目统一 `summary / overview / details` 数据结构 |
| 后端 | 听读写口返回结构尽量统一，减少前端分支判断 |
| 阅卷系统 | 主观题点评与评分维度建议独立结构返回 |
